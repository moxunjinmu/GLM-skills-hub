## 8. 可访问性与一致性约束（全局）
- 触控区域 ≥ 44x44px，焦点可见
- 所有交互元素有 hover/active 状态
- 玻璃卡片透明度需确保文字对比度 ≥ 4.5:1
- 中英文本长度差异需兼容换行/截断（2 行为上限）
