```
│ 重置/应用      │ 分页                                           │
```

### 1.2 平板端布局（768-1279）
```
顶部导航
面包屑 / 标题
筛选折叠按钮（抽屉形式）
技能卡片网格（2 列）
分页
```

### 1.3 移动端布局（<= 767）
```
顶部导航（简化）
搜索栏
筛选按钮（抽屉）
技能卡片列表（1 列）
分页（简化）
```

### 1.4 组件组合
- 导航：顶栏 + 语言切换 + 登录入口
- 搜索：顶部搜索框 + 标签推荐
- 筛选：筛选面板（多选、折叠）
- 列表：Skill 卡片（标题/平台/Stars/更新时间/试用按钮）
- 分页：页码 + 上下页

```
┌──────────────────────────────────────────────────────────────┐
│ 顶部导航                                                     │
├──────────────────────────────────────────────────────────────┤
```

```
│ 历史记录（右侧抽屉/浮层）                                    │
└──────────────────────────────────────────────────────────────┘
```

### 3.2 平板端布局（768-1279）
```
顶部导航
状态栏

```

### 3.3 移动端布局（<= 767）
```
顶部导航
```

### 3.4 组件组合

## 4. 统一交互规范
- 列表页：筛选改变 → 顶部显示筛选条件标签
- 详情页：切换 Tab 保持滚动位置可见
- 试用页：提示 Token 实时扣减，余额不足时提示升级入口
