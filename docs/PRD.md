# Skills 集合站 PRD（MVP + 二期）

## 1. 项目概述
- 产品名称：Skills 集合站（暂定）
- 产品定位：面向多平台 Skill 的中文化聚合目录与试用平台
- 核心价值：
  - 中文介绍 + 中文使用指南（降低入门门槛）
  - 在线试用（免费积分）
  - 热门趋势追踪（多源热度融合）
- MVP 范围：仅收录「Skill」能力（不含插件、MCP、Prompt 等）

## 2. 目标用户
- 主要面向国内用户，兼顾国际访问
- 个人开发者/AI 工具爱好者
- 企业研发、运营、增效团队
- Skills 贡献者与维护者
- 平台运营与审核人员

## 3. 平台覆盖范围
- Claude / Codex / Cursor / OpenCode / Gemini / Coze / 其他支持 Skill 的平台
- MVP 只收录开源仓库

## 4. 需求总结（MVP）
### 4.1 功能模块
- 收录与同步
  - MVP 聚合来源：仅使用 GitHub API
    - GitHub 仓库聚合（awesome-claude-skills、baoyu-skills、myclaude、anthropics/skills 等）
  - 自动抽取：从仓库解析 SKILL.md / README / 元数据
  - 更新策略：MVP 每日定时更新；支持手动刷新

- Skills 详情页
  - 中文简介 / 英文原文
  - 中文使用指南（步骤、参数、示例）
  - 原仓库信息（仓库链接、许可证、更新日期、Stars/Forks）
  - 适用场景、限制与风险提示

- 在线试用（MVP：热门技能）
  - Web 对话式试用
  - 仅开放部分热门 Skill
  - 试用记录可复现

- 分类与搜索
  - 多维分类（场景/平台/技术栈/复杂度/安全等级）
  - 搜索 + 标签筛选

- 热门榜单
  - 基于多源热度
  - 维度：今日/本周/本月/历史

- 提交与审核
  - 收录入口（仓库链接 + 基本信息）
  - 自动扫描：许可证、可疑行为提示
  - 人工审核后上线

- i18n 国际化
  - 首发语言：中文 + 英文
  - 默认：自动识别浏览器语言 + 手动切换

### 4.2 用户体系（MVP）
- 基础注册/登录
- 收藏与历史
- 免费积分

## 5. 试用积分建议
- 初始积分：500 分（按 10 元试用额度折算）
- 计费建议：
  - 定价基准：1 亿 tokens = 599 元
  - 折算规则：10 元试用额度 = 500 积分
  - Token 换算：
    - 1 分 ≈ 3,339 Token（按上述基准折算，取整可用 3,300 Token）
  - 试用消耗建议：每次试用 50 分（目标：约 10 次）
  - 单次试用 Token 上限：约 166,000 Token（50 分 × 3,300 Token）
- 运营策略：签到/分享/贡献补充积分

## 6. 热门榜单建议（权重示例）
**MVP（仅 GitHub）：简单 Stars + 活跃度**
- Stars（权重 0.70）
- 活跃度（权重 0.30）：最近更新频率/最近提交时间/近 30 天活跃信号（以 GitHub API 可获得字段为准）

**二期（多源热度，固定权重）：**
- GitHub 维度（权重 0.45）：Stars、Forks、最近更新频率
- 搜索引擎（权重 0.25）：Google + 百度搜索热度
- 社交媒体（权重 0.20）：Twitter/X、微信公众号
- 站内数据（权重 0.10）：收藏、试用次数

**时间衰减：**对近 7/30 天加权，降低历史累积偏差

## 7. 内容生产与翻译
- 中文介绍与使用指南由模型生成 + 人工审校
- 模型方案：
  - GLM（智谱）或 Qwen API
- 术语库：统一平台与技能名称翻译
- 标准化输出：标题、摘要、使用步骤、注意事项、示例
- 翻译与 i18n 流程：
  - 翻译结果入库（i18n 字段）
  - Skill 更新时触发增量翻译更新
  - 优先复用历史翻译，减少接口调用成本

## 8. 数据模型（核心字段建议）
- Skill
  - id / name / aliases
  - platform（Claude/Codex/Cursor/...）
  - description_cn / description_en
  - guide_cn / guide_en
  - tags / categories
  - repo_url / license / stars / forks / last_update
  - risk_level / status（草稿/审核中/上线/下线）
  - popularity_score
- Trial
  - skill_id / user_id / tokens_used / created_at

### 9.3 视觉风格与设计系统（高保真）
- 风格关键词：现代 SaaS / 轻玻璃质感 / 清爽可信 / 数据感
- 视觉模式：浅色为主（玻璃卡片在浅色背景上）
- 颜色体系（可调优先保持对比度）
  - 主色：#3B82F6（Primary）
  - 次色：#60A5FA（Secondary）
  - 强调 CTA：#F97316（CTA）
  - 背景：#F8FAFC（Background）
  - 文字：#1E293B（Text）
  - 辅助灰：#64748B（Muted）
  - 边框：#E2E8F0（Border）
- 字体与排版
  - 英文标题/正文：Plus Jakarta Sans
  - 中文回退：PingFang SC / Noto Sans SC / Microsoft YaHei
  - 标题层级：H1 40px / H2 28px / H3 20px
  - 正文：16px / 行高 1.6
- 栅格与尺寸
  - 容器宽度：1200-1280px
  - 栅格：12 列 / 间距 24px
  - 圆角：卡片 12px / 按钮 10px / 输入框 10px
  - 阴影：卡片轻阴影 + hover 强化
- 玻璃卡片规范
  - 背景：白色 80%-90% 透明度
  - 边框：1px rgba(255,255,255,0.6)
  - 模糊：10-16px
  - 需确保文本对比度 >= 4.5:1
- 组件库（高保真统一组件）
  - Button：Primary / Secondary / Ghost / Disabled
  - Input：搜索框（圆角胶囊）/ 标准输入框
  - Card：Skill 卡 / 排名卡 / 信息卡
  - Tag/Badge：平台/场景/风险等级
  - Tabs：详情页内容切换
  - Table/List：榜单与历史记录
  - Tooltip：指标解释
- 交互与动效
  - Hover：阴影增强、边框高亮
  - Transition：150-250ms
  - Focus：明确焦点环
  - Loading：Skeleton + 按钮禁用态

- 阶段 1（MVP）：
  - 收录 200+ 热门 Skill

  - 每日定时更新
  - 搜索/分类/详情/试用（部分）/榜单/i18n
  - 日活目标：几百
- 阶段 2：
  - 私有技能库 + 私有试用
  - 更完整的评论体系
  - 企业协作与权限
